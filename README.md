# Mi proyecto final Automatización de Pruebas

## Propósito
Este proyecto tiene como objetivo desarrollar un framework de automatización de pruebas utilizando Python, Selenium y Requests.

## Tecnologías Utilizadas
- Python
- Selenium WebDriver
- Pytest
- Biblioteca Requests

## Estructura del Proyecto
- `pages/`: Contiene las clases para cada página de la aplicación.
- `tests/`: Contiene los casos de prueba automatizados.
- `utils/`: Funciones utilitarias para cargar datos y manejar logs.
- `reports/`: Reportes generados por pytest.
